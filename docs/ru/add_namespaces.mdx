Для обеспечения точного контроля над службами в сети вы можете ограничить область, которую сеть может контролировать, управляя пространствами имен. Службы могут быть добавлены и управляться в пространстве имен только в том случае, если пространство имен управляется сервисной сетью.

**Примечание**: Вы можете исключить пространство имен из управления, удалив его, но только пространства имен без служб могут быть удалены.

### Шаги

1. В левой навигационной панели нажмите **Service Mesh** > **Meshes**.

2. Нажмите на ***Имя сервисной сети***, для которого вы хотите установить политику.

3. Прокрутите вниз до раздела **Namespaces**, выберите кластер и нажмите **Управлять пространством имен**.

4. Выберите пространство имен в подкластер и нажмите **Подтвердить**.
