# Введение

## Расширенные операции с сайдкаром

Расширенные операции с сайдкаром предоставляют детальный контроль над поведением прокси Istio. Эти функции:

- Оптимизация производительности сервисной сетки
- Возможность целенаправленной диагностики
- Предложение временных путей восстановления сервиса
- Поддержание гибкости управления

Основная ценность: Увеличенный контроль над компонентами сервисной сетки

## Преимущества

- ​**Без разрушений**: Режим обхода без перезапуска пода
- ​**Точность**: Специфичный для порта/диапазона IP проброс
- ​**Диагностика**: Корректировка уровня логирования в режиме выполнения
- ​**Резервное копирование**: Полная возможность удаления сайдкара

## Сценарии

- ​**Оптимизация промежуточного программного обеспечения**
  Освободите трафик Redis от обработки сайдкаром

- ​**Диагностика производительности**
  Временно обойти сайдкар для изоляции проблем

- ​**Поддержка отладки**
  Включить отладочные логи для конкретных служб

- ​**Совместимость с устаревшими системами**
  Удалить сайдкар для требований к регистрационным заголовкам с учётом регистра

## Ограничения

- Режим обхода требует Kubernetes 1.23+
- Конфигурация проброса инициирует перезапуск развертывания
- Удаленный сайдкар отключает политики безопасности
- Синхронизация логов в многокластерной среде увеличивает задержку
