# Отключение уведомлений

## Правила отключения уведомлений

Установив отключение для определенной политики уведомлений, вы можете контролировать, чтобы все правила в рамках политики уведомлений не отправляли уведомления о срабатывании на целевые адреса уведомлений в течение установленного периода тишины.

**Сценарии использования**: Во время обновлений платформы или технического обслуживания многие статусы ресурсов будут аномальными, и соответствующие уведомления будут срабатывать. Операционному персоналу будет часто приходить уведомление о срабатывании до завершения обновления или технического обслуживания. Установив отключение для политики уведомлений, можно избежать таких ситуаций.

**Примечание**: Установка отключения не влияет на срабатывание уведомлений, и вы все еще можете просматривать статус уведомлений на странице **Политики уведомлений** / **Внедрение уведомлений**. Поддерживаются постоянное отключение и настройки пользовательского времени отключения; как только время окончания пользовательского отключения истекает, настройка отключения автоматически снимается.

**Шаги**

1. В левой панели навигации нажмите **Уведомления** > **Политики уведомлений**.

2. Нажмите на <img src="../../../en/metrics_monitoring/assets/three_point.png" style={{display: "inline", verticalAlign: "middle"}} /> рядом с политикой уведомлений, которую нужно отключить > **Установить отключение**.

3. Нажмите, чтобы включить переключатель **Отключение уведомлений**.

   **Совет**: Этот переключатель используется для контроля, будет ли настройка отключения эффективной. Чтобы отменить отключение, просто выключите переключатель.

4. Обратитесь к следующим инструкциям, чтобы настроить соответствующие параметры.

   **Совет**: Если не выбрано имя ресурса, по умолчанию используется **Любой**, т.е. при **удалении/добавлении** ресурсов впоследствии платформа будет **удалять отключение/добавлять отключение** к политике уведомлений; если выбрано **Выбрать все**, это применяется только к в настоящее время выбранным ресурсам, последующие **удаление/добавление** ресурсов не будет обрабатываться.

   | Параметр            | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
   | -------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Цель отключения**   | Объект ресурса, на который воздействует настройка отключения.<br /> Вы можете выбрать некоторые вычислительные компоненты, связанные с политикой уведомлений, максимум 30; при выборе **Все вычислительные компоненты** цель отключения включает будущие компоненты того же типа, добавленные в текущем пространстве имен.                                                                                                                                                                |
   | **Продолжительность отключения** | Временной диапазон отключения уведомлений. Уведомление вступает в безмолвное состояние с момента начала времени отключения, и если политика уведомлений все еще находится в состоянии уведомления или триггерит уведомление после истечения времени отключения, уведомления о срабатывании возобновятся.<br />**Постоянное**: После установки отключения, уведомление остается в безмолвном состоянии до тех пор, пока политика уведомлений не будет удалена.<br />**На заказ**: Установите время начала и окончания отключения по желанию, с минимальным интервалом 5 минут. |

5. Нажмите **Установить**.

   **Совет**: С момента установки отключения до начала отключения статус отключения политики уведомлений будет **Ожидание отключения**, в течение которого, если правило в политике срабатывает, уведомления будут отправлены нормально; с момента начала отключения до его завершения статус отключения политики уведомлений составляет **Отключение**, в течение которого, если правило в политике срабатывает, уведомления не будут отправлены.
