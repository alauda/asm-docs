# Обзор функций

## Конфигурация прокси Envoy

- **Прямой контроль конфигурации Envoy**\
  Позволяет детальную модификацию настроек прокси Envoy, включая вставку фильтров, создание слушателей/кластера и настройки на уровне полей. Позволяет настраивать параметры за пределами стандартных сетевых API Istio.

- **Настройка под конкретные версии**\
  Поддерживает оптимизацию параметров на уровне прокси для конкретных версий Envoy, включая настройки пула соединений и конфигурации таймаута.
