# Обновление

При обновлении версии Istio в сервисной сетке, чтобы избежать влияния на нормальную работу бизнеса, пожалуйста, создайте EnvoyFilter, совместимый с более высокой версией Istio, перед обновлением. После обновления сервис будет автоматически привязан к EnvoyFilter, совместимому с обновленной версией.
