# Плагин кластера Alauda Service Mesh Essentials

Плагин кластера Alauda Service Mesh Essentials предоставляет интерфейс пользователя, расширенный сервер API и глобальный контроллер asm.

## Загрузка

:::info
Плагин кластера Alauda Service Mesh Essentials можно получить из Alauda Cloud.

Пожалуйста, свяжитесь со службой поддержки клиентов для получения дополнительной информации.
:::

## Загрузка

Необходимо загрузить плагин кластера Alauda Service Mesh Essentials в **глобальный** кластер ACP.

### Процедура

<Steps>
  ### Загрузка инструмента violet

  Сначала нужно загрузить инструмент `violet`, если он не присутствует на машине.

  В **Управлении платформой** консоли ACP:

  1. Нажмите **Marketplace / Загрузить пакеты**.
  2. Нажмите **Загрузить инструмент упаковки и листинга**.
  3. Найдите правильную ОС / архитектуру процессора в разделе **Исполнительная среда**.
  4. Нажмите **Загрузить**, чтобы загрузить инструмент `violet`.
  5. Выполните `chmod +x ${PATH_TO_THE_VIOLET_TOOL}`, чтобы сделать инструмент выполняемым.

  ### Загрузка пакета

  Затем выполните следующую команду, чтобы загрузить плагин кластера **Alauda Service Mesh Essentials** в глобальный кластер ACP:

  ```bash
  # [!code callout:5]
  violet push \
      ${PKG_NAME} \
      --platform-address=${PLATFORM_ADDRESS} \
      --platform-username=${PLATFORM_ADMIN_USER} \
      --platform-password=${PLATFORM_ADMIN_PASSWORD} \
      --clusters=global
  ```

  <Callouts>
    1. Выполните `violet push`, чтобы загрузить плагин кластера.
    2. `${PKG_NAME}` - это имя архива плагина кластера **Alauda Service Mesh Essentials**.
    3. `${PLATFORM_ADDRESS}` - это адрес вашей платформы ACP.
    4. `${PLATFORM_ADMIN_USER}` - это имя пользователя администратора платформы ACP.
    5. `${PLATFORM_ADMIN_PASSWORD}` - это пароль администратора платформы ACP.
  </Callouts>
</Steps>

## Установка

### Процедура

В **Управлении платформой** консоли ACP:

1. Выберите **глобальный** кластер из выпадающего списка **Кластер** в верхней части консоли.
2. Нажмите **Marketplace / Плагин кластера**.
3. Найдите **Alauda Service Mesh Essentials**.
4. Нажмите **Установить**.

### Проверка

Подтвердите, что плитка **Alauda Service Mesh Essentials** отображает одно из следующих состояний:

- `Установка`: установка в процессе; дождитесь, пока это состояние изменится на `Установлено`.
- `Установлено`: установка завершена.
