平台上的功能可以根据其成熟度分为 Alpha 版本、Beta 版本和 GA（正式发布）版本。

Beta 版本和 GA（正式发布）版本的功能要么是稳定的，要么处于稳定迭代的过程中。专业团队提供支持和服务。您可以根据实际需求在生产环境中使用相应的功能。

Alpha 版本的功能可能具有缺陷或严重的错误，默认情况下不在生产环境中启用。仅推荐在设置用于体验或测试平台功能的环境中使用。

为了帮助您区分 Alpha 版本的功能，在文档的左侧目录结构中，处于 Alpha 版本的模块或特定功能将在导航菜单中标记为 `Alpha`。例如：EnvoyFilter（Alpha）。

以下表格描述了平台功能的成熟度级别及其推荐的使用范围。

| 版本       | 成熟度描述                                                                                                                                                                                                                                                                                                                                                                                                                                                      | 推荐使用范围                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| :--------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Alpha**  | <ul><li>可能存在缺陷和错误，启用后可能会遇到严重的 bug。</li><li>支持的功能随时可能在不通知的情况下被删除。</li><li>API、产品场景、交互过程等的变化可能会导致兼容性问题，但不会在随后发布的版本中具体声明。</li><li>如果依赖的 Kubernetes 功能处于 Alpha 版本，相应的平台功能只能处于 Alpha 版本。</li></ul> | Alpha 版本的功能在私有环境部署中默认禁用。如有必要，可以通过配置相关参数启用。<br />由于存在 bug 风险和可能缺乏长期支持，建议仅在短期功能体验或测试环境中使用，例如概念验证（POC）环境。                                                                                                                                                                                                                     |
| **Beta**   | <ul><li>可能存在缺陷，但所有支持的功能不会轻易删除。</li><li>API 参数、产品设计细节等，可能会随着版本迭代而变化。在这种情况下，我们将提供迁移到下一个版本的说明。在执行编辑操作时需要小心，依赖于该功能的应用程序可能需要禁用。</li></ul>                                                                                                                                                                                                                                         | Beta 版本的功能在默认情况下启用，可通过配置相关参数在部署过程中禁用。<br />后续版本中可能存在不兼容的更改，因此建议仅在非关键业务运营环境中使用。如果有多个集群可以独立升级，则可以放宽在生产环境中的使用限制。                                                                                                                                                                                                                                                                   |
| **GA**     | <ul><li>功能的稳定版本，将出现在许多后续版本中。</li><li>API 参数、产品设计细节等可能会随着长期支持版本而变化。在这种情况下，我们将提供迁移到下一个版本的说明。</li></ul>                                                                                                                                                                                                                                                                                         | 适用于所有环境，包括生产环境。                                                                                                                                                                                                                                                                                                                                                                                                                                    |
