平台上的功能可根据其成熟度分为 Alpha 版本、Beta 版本和 GA（一般可用性）版本。

Beta 版本和 GA（一般可用性）版本的功能要么是稳定的，要么正处于稳定化迭代过程中。专业团队提供支持和服务。您可以根据实际需求在生产环境中使用相应的功能。

Alpha 版本的功能可能存在缺陷或严重的错误，且默认不在生产环境中启用。仅建议在为体验或测试平台功能而设置的环境中使用。

为了帮助您区分 Alpha 版本的功能，在文档的左侧目录结构中，处于 Alpha 版本的模块或特定功能将在导航菜单中标记为 `Alpha`。例如：EnvoyFilter (Alpha)。

以下表格描述了平台功能的成熟度等级及其推荐使用范围。

| 版本       | 成熟度说明                                                                                                                                                                                                                                                                                                                                                                                                                                                                           | 推荐使用范围                                                                                                                                                                                                                                                                                                                                                                                             |
| :-------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Alpha** | <ul><li>可能存在缺陷和错误，启用后可能会遇到严重的错误。</li><li>受支持的功能可能会在没有任何通知的情况下被删除。</li><li>API、产品场景、交互流程等的变化可能会导致兼容性问题，但不会在后续发布中专门声明。</li><li>如果依赖的 Kubernetes 功能处于 Alpha 版本，则相应的平台功能只能处于 Alpha 版本。</li></ul>                                         | Alpha 版本的功能在私有环境部署中默认禁用。必要时可以通过配置相关参数启用。<br />由于存在错误风险和可能缺乏长期支持，建议仅在短期功能体验或测试环境中使用，例如概念验证（POC）环境。                                                                                |
| **Beta**  | <ul><li>可能存在缺陷，但所有受支持的功能不会轻易删除。</li><li>API 参数、产品设计细节等可能会随着版本迭代而变化。在这种情况下，我们将提供迁移到下一个版本的说明。执行编辑操作时需要谨慎，依赖该功能的应用可能需要被禁用。</li></ul>                                                                                           | Beta 版本的功能在部署时默认启用，并可以通过配置相关参数禁用。<br />后续版本中可能会出现不兼容的更改，因此建议仅在非关键业务操作环境中使用。如果存在多个可以独立升级的集群，此限制可以放宽，以便在生产环境中使用。                                                                               |
| **GA**    | <ul><li>功能的稳定版本，在后续许多版本中都会出现。</li><li>API 参数、产品设计细节等可能会随着长期支持版本而变化。在这种情况下，我们将提供迁移到下一个版本的说明。</li></ul>                                                                                                                                                                                                                                                | 适用于所有环境，包括生产环境。                                                                                                                                                                                                                                                                                                                                                                           |
