# 警报静默

## 警报静默中的规则

通过为指定的警报策略设置静默，您可以控制该警报策略下的所有规则在设定的静默期间不向通知目标发送警报通知消息。

**使用场景**：在平台升级或维护期间，许多资源状态将会异常，相关的警报将被触发。在升级或维护完成之前，运维人员会频繁收到警报通知。通过为警报策略设置静默，可以避免这种情况的发生。

**注意**：设置静默不影响警报的触发，您仍然可以在 **警报策略** / **执行警报** 页面查看警报状态。支持永久静默和自定义时间静默设置；一旦自定义时间静默的结束时间到期，静默设置将自动清除。

**步骤**

1. 在左侧导航栏中，点击 **警报** > **警报策略**。

2. 点击要进行静默的警报策略旁的 <img src="../assets/three_point.png" style={{display: "inline", verticalAlign: "middle"}} /> > **设置静默**。

3. 点击开启 **警报静默** 开关。

   **提示**：此开关用于控制静默设置是否生效。如需取消静默，只需将开关关闭即可。

4. 参照以下说明配置相关参数。

   **提示**：如果未选择资源名称，默认为 **任意**，即后续 **删除/添加** 资源时，平台将对警报策略进行 **删除静默/添加静默**；如果选择了 **全选**，则仅适用于当前选择的资源，后续的 **删除/添加** 资源将不予处理。

   | 参数                | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
   | ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **静默目标**       | 静默设置的目标资源对象。<br />您可以选择与警报策略相关的一些计算组件，最多可选30个；当选择 **所有计算组件** 时，静默目标包括在当前命名空间下添加的同类型的未来组件。                                                                                                                                                                                                                                                                                              |
   | **静默时长**      | 警报静默的时间范围。警报从静默开始时间进入静默状态，如果在静默结束时间后，警报策略仍处于警报状态或触发警报，则会恢复通知。<br />**永久**：在设置静默后，警报将一直处于静默状态，直到删除该警报策略。<br />**自定义**：可自定义设置静默的开始和结束时间，最小间隔为5分钟。 |

5. 点击 **设置**。

   **提示**：从静默设置时间到静默开始时间，警报策略的静默状态为 **静默等待**，在此期间，如果策略中的规则触发警报，通知将正常发送；从静默开始到静默结束，警报策略的静默状态为 **静默中**，在此期间，如果策略中的规则触发警报，将不发送任何通知。
