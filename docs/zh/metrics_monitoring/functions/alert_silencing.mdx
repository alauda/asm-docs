# 警报静默

## 警报静默中的规则

通过为指定的警报策略设置静默，您可以控制该警报策略下的所有规则在设定的静默期内不向通知目标发送警报通知消息。

**使用场景**：在平台升级或维护期间，许多资源状态将会异常，相关的警报将会被触发。运营人员在升级或维护完成之前会频繁收到警报通知。通过为警报策略设置静默，可以避免此类情况的发生。

**注意**：设置静默并不影响警报的触发，您仍然可以在**警报策略** / **实施警报**页面查看警报状态。支持永久静默和自定义时间静默设置；一旦自定义静默的结束时间过期，静默设置将会自动清除。

**步骤**

1. 在左侧导航栏中，点击**警报** > **警报策略**。

2. 点击要静默的警报策略旁边的 <img src="../assets/three_point.png" style={{display: "inline", verticalAlign: "middle"}} /> > **设置静默**。

3. 点击打开**警报静默**开关。

   **提示**：该开关用于控制静默设置是否生效。要取消静默，只需关闭开关即可。

4. 请参阅以下说明配置相关参数。

   **提示**：如果没有选择资源名称，默认值为**任意**，即在后续**删除/添加**资源时，平台将**删除静默/添加静默**到警报策略；如果选择**全部选择**，则仅适用于当前选定的资源，后续的**删除/添加**资源将在此情况下不被处理。

   | 参数                | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
   | ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
   | **静默目标**       | 静默设置所针对的资源对象。<br />您可以选择与警报策略相关的一些计算组件，最多可选择30个；选择**所有计算组件**时，静默目标包括当前命名空间下同类型未来添加的组件。                                                                                                                                                                                                                                                                                                                      |
   | **静默持续时间**   | 警报静默的时间范围。警报从静默开始时间进入静默状态，如果警报策略在静默结束时间后仍处于警报状态或触发警报，警报通知将恢复。<br />**永久**：静默设置后，警报将保持静默状态，直到警报策略被删除。<br />**自定义**：自定义设置静默的开始和结束时间，最小间隔为5分钟。 |

5. 点击**设置**。

   **提示**：从设置静默时刻到静默开始时，警报策略的静默状态为**静默等待**，在此期间，如果策略中的某个规则触发警报，通知将正常发送；从静默开始到静默结束，警报策略的静默状态为**静默中**，在此期间，如果策略中的某个规则触发警报，将不会发送任何通知。
