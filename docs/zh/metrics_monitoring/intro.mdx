# 引言

## 指标监控

该平台提供了全面的流量监控指标，允许用户从多个角度分析服务的流量性能和质量。

此外，该平台还提供了JVM监控，展示了通过OpenTelemetry Java Agent收集的JVM（Java虚拟机）指标。这些指标包括GC（垃圾收集）、堆内存、非堆内存、JVM线程、Java类等。您可以使用监控图表中呈现的实时监控数据进行性能分析，快速识别内存泄漏、线程异常和其他问题。

## 告警

在云原生环境中，告警是一种关键的监控机制，当系统出现潜在问题时，它能够及时通知运维人员或开发人员。该机制帮助团队快速响应并解决可能影响应用性能、可用性或安全性的问题。云原生环境通常是动态且分布式的，包含大量的微服务和容器，因此有效的告警系统成为维护系统稳定性的重要组成部分。
