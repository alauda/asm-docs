# 介绍

## 监控指标

该平台提供全面的流量监控指标，使用户能够从多个角度分析服务流量的性能和质量。

此外，该平台还提供JVM监控，显示由OpenTelemetry Java Agent收集的JVM（Java虚拟机）指标。指标包括GC（垃圾回收）、堆内存、非堆内存、JVM线程、Java类等。您可以使用监控图表中呈现的实时监控数据进行性能分析，快速识别内存泄漏、线程异常和其他问题。

## 告警

在云原生环境中，告警是一个关键的监控机制，当系统中出现潜在问题时，能够及时通知运维人员或开发人员。该机制帮助团队快速响应并解决可能影响应用性能、可用性或安全性的问题。云原生环境通常是动态和分布式的，包含大量的微服务和容器，因此有效的告警系统是维护系统稳定性的重要组成部分。
