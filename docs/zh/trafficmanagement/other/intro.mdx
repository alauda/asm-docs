# 介绍

## 高级边车操作

高级边车操作提供了对 Istio 代理行为的细粒度控制。这些功能：

- 优化服务网格性能
- 实现有针对性的故障排查
- 提供临时服务恢复路径
- 维护治理灵活性

核心价值：增强对服务网格组件的控制

## 优势

- **非干扰性**：可在不重启 Pods 的情况下使用旁路模式
- **精准性**：特定端口/IP 范围的直通
- **诊断**：运行时日志级别调整
- **回退**：完全移除边车的能力

## 场景

- **中间件优化**
  使 Redis 流量免于边车处理

- **性能故障排查**
  暂时旁路边车以隔离问题

- **调试支持**
  为特定服务启用调试日志

- **遗留兼容性**
  移除边车以满足大小写敏感的头部要求

## 限制

- 旁路模式要求 Kubernetes 1.23 以上版本
- 直通配置会触发部署重启
- 移除边车会禁用安全策略
- 多集群日志同步增加延迟
