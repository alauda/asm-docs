# 介绍

## 服务的全局流量限制介绍

全局流量限制模块通过智能流量控制保护服务免受高并发场景下的过载。它支持：

- 在流量高峰期间防止服务崩溃
- 通过多维度规则实现精细化的访问控制
- 通过影子模式分析实现安全的容量规划

## 优势

- **高并发保护**：保持服务在资源限制内的稳定性
- **精确控制**：基于 API 路径/方法/头部的流量限制
- **无风险测试**：影子模式模拟，不影响生产环境
- **实时可视化**：对限流效果进行可视化监控

## 场景

- **API 过载保护**  
  保护关键 API 免受突发流量激增

- **敏感操作限流**  
  对写入/删除操作实施严格限制

- **多租户配额管理**  
  在客户端群体中实施公平使用政策

## 限制

- 需要预先配置的 Redis 集群集成
- 目标服务强制注入 sidecar
- 每个服务最多只能有 1 个全局策略
- 正则匹配仅支持 RE2 语法
- 影子模式数据保留期：7 天
