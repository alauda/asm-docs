# 功能总览

## Envoy Proxy 配置

- **直接控制 Envoy 配置**\
  允许对 Envoy 代理设置进行细致的修改，包括过滤器插入、监听器/集群创建以及字段级别的调整。允许超出 Istio 标准网络 API 的自定义。

- **版本特定调优**\
  支持针对特定 Envoy 版本的代理级参数优化，包括连接池设置和超时配置。
