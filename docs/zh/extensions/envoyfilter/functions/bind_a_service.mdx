# 绑定服务

通过将创建的 EnvoyFilter 绑定到服务，可以将该 EnvoyFilter 的功能分配给所在集群中的服务。

## 前提条件

EnvoyFilter 的 **Istio 兼容版本** 包括该服务网格的 Istio 版本。

## 操作步骤

1. 在左侧导航栏中，点击 **服务网格** > **EnvoyFilter**。

2. （可选）在顶部导航栏中，点击切换到 **服务网格**。

3. 点击要绑定到服务的 ***EnvoyFilter*** 名称。

4. 在 **绑定服务** 标签下，点击 **绑定服务**。

5. 参考以下说明配置相关参数。

   - **命名空间**：EnvoyFilter 在集群中所在的命名空间。

     **提示**：在配置过程中，您可以切换命名空间并单独配置它们。切换命名空间不会影响已配置的数据。例如，在命名空间 `ns 1` 中，如果选择了 3 个服务，然后切换到命名空间 `ns 2` 并选择了 4 个服务，当您执行绑定操作时，所有 7 个选定的服务将被绑定。

   - **清除选择**：清除 **已选择** 服务列表中的服务。清除后，**选择服务** 列表中的所有服务将显示为未选择状态。

   - **删除已选择服务**：已从“已选择”服务列表中删除的服务将显示为“选择服务”列表中的未选择状态。

6. 确认信息正确后，点击 **绑定**。
