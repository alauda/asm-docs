本快速入门旨在指导您在服务网格上部署和配置应用程序。通过遵循本指南，您将学习如何部署应用程序并为基本场景配置服务网格及其组件。

在本快速入门中，您将体验以下场景：

- 部署 Bookinfo 示例应用程序
- 通过 Istio Ingress Gateway 访问 Bookinfo
- 探索可观察性功能

在开始之前，请确保您已完成以下步骤：

- 阅读 [关于服务网格](overview/index.html)。
- [创建服务网格](install/create_service_mesh/index.html)。
