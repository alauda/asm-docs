:::tip

If the module needs to be deployed separately, deployment documentation must be provided; otherwise, it can be omitted.

The deployment documentation should include deployment procedures, deployment configurations, uninstallation processes, etc. If the content of the deployment documentation is extensive, it can be converted into a directory and split into multiple documents for introduction.

You can visit [Deployment](https://product-doc-guide.alauda.cn/template/installation/example.html) to see examples of the corresponding documentation.

:::
