This quick start is designed to walk you through the deployment and configuration of applications on a Service Mesh. By following this guide, you will learn how to deploy applications and configure the Service Mesh and its components for basic scenarios.

In this quick start, you will experience the following scenarios:

- Deploy the Bookinfo sample application
- Access Bookinfo through the Istio Ingress Gateway
- Explore observability features

Before you begin, please make sure you have completed the following steps:

- Read the [About Service Mesh](../overview/index.mdx).
- [Creating a Service Mesh](../install/create_service_mesh/index.mdx).
