# Features Overview

## Envoy Proxy Configuration

- **Direct Envoy Configuration Control**  
  Enables granular modification of Envoy proxy settings, including filter insertion, listener/cluster creation, and field-level adjustments. Allows customization beyond Istio's standard networking APIs.

- **Version-Specific Tuning**  
  Supports proxy-level parameter optimization for specific Envoy versions, including connection pool settings and timeout configurations.
