# 权限说明

取消代码块，替换 `function` 参数的值为需要展示的 `FunctionResource` 资源名称:
使用如下 `<K8sPermissionTable \>` 组件自动生成权限表格，具体使用方式参见 [自动生成权限表格指南](https://product-doc-guide.alauda.cn/guide/auto_gen_permissions.html)。

```mdx
<K8sPermissionTable functions={['<functionName1>', '<functionName2>']} />
```
