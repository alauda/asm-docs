# Create Service Mesh

## Prerequisites

Ensure the following operators are installed before proceeding:

* `Alauda Service Mesh Essentials` cluster plugin, please visit [Alauda Service Mesh Essentials Cluster Plugin](../asm-essentials.mdx)) for installation.
* `Alauda Service Mesh` operator, please visit [Install Operator](../install-operator.mdx) for how to install an operator.
* `Flagger` operator, please visit [Install Operator](../install-operator.mdx) for how to install an operator.
* `Jaeger` operator, please visit [Install Operator](../install-operator.mdx) for how to install an operator.
* `OpenTelemetry` operator, please visit [Install Operator](../install-operator.mdx) for how to install an operator.
